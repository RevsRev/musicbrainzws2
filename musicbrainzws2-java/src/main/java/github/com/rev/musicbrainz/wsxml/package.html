A parser for the Music Metadata XML Format (MMD).

This module contains {@link github.com.rev.musicbrainz.wsxml.impl.JDOMParser}, which parses the Music Metadata XML Format (MMD) returned by the MusicBrainz webservice. There are also DOM helper functions in this module used by the parser which probably aren't useful to users.

You can implement your own {@link github.com.rev.musicbrainz.wsxml.MbXmlParser} and inject it through one of the {@link github.com.rev.musicbrainz.Query} constructors to use it.

@see github.com.rev.musicbrainz.wsxml.impl.JDOMParser
