Classes for interacting with the MusicBrainz XML web service.

Implementations of the {@link github.com.rev.musicbrainz.webservice.WebService} interface talk to a server implementing the MusicBrainz XML web service. They mainly handle URL generation and network I/O. Extend {@link github.com.rev.musicbrainz.webservice.DefaultWebService} if maximum control is needed.

The {@link github.com.rev.musicbrainz.Query} class provides a convenient interface to the most commonly used features of the web service. By default it uses {@link github.com.rev.musicbrainz.webservice.impl.HttpClientWebService} to retrieve data and {@link github.com.rev.musicbrainz.wsxml.impl.JDOMParser} to parse the responses. The results are object trees using the MusicBrainz domain model.
